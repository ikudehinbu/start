HTML TAGS

Web pages are designed using HTML in which the browser interprets in order to display on the web page for user to see. HTML consist of text that are wrapped up within tags which helps the browser to know how to display the the text on the web page.

HTML TAGS AND THEIR USES

There are various HTML tags that are used in for web development. Below are some HTML tags and there uses:
  • Brake line tag (br): This HTML tag is used for lining breakage. Other words or sentences after this tag continues on another line.
    Example:
    I am going home now.
    Send my regrads to your children.
  • Center tag (center): This HTML tag is used to put words ot text at the center.
    Example:
    I want to walk in the middle of you two.
  • Preserve format tag (pre): This HTML tag is used to preserve the format in which a text is written in the HTML document when its been displayed by the browser on the web page.
    Example: 
               what are you      doing    now
               i am going home
  • Underline tag (u): This tag is used to underline text.
    Example:
    when you get home, call me.
  • Italic tag (i): This tag is used to diplay text in italic form.
    Example:
    I am doing my project at Jimmy's place.
  • Bold tag (b): This tag is used to display text in bold form.
    Example:
    I am very scared of the dark
  • Delete tag (del): This tag is used to display text in delete form.
    Example:
    I will call you when i get arrive home.
  • Insert tag (ins): This tag is used to display text in inserted form.
    Example:
    Daniel is on his way to the party.
  • Superscript tag (sup): This tag is used to display text in superscript form.
    Example:
    Hundred raised to the power of too (1002) equals ten thousand.
  • Subscript form (sub): This tag is used to display text in subscript form.
    Example:
    The formula or water is H2O.
  • Acronym tag (acronym): This tag is used to display text in acronym form.
    Example:
    HTML is the acronym for Hyper text Markup language.
  • Mark tag (mark): This tag is used to display text in marked form.
    Example:
    John is a very brilliant child
  • Quote tag (blockquote): This tag is used to qoute text.
    Example:
    A wise man once said
    "Cut your coat according to your size."
  • Citation tag (cite): This is used to cite a text.
    Example:
    Obedience is better than sacrifice
  • Abbreviation tag (abbr): This is used for abbreviation of words.
    Example:
    E.g is the abbrevation of Example.

  • Large tag (big): This is used to a make a test one font larger than other text.
    Example:
    I am bigger than the others.
  • Small tag (small): This is used to a make a test one font smaller than other text.
    Example:
    I am smaller than the others.
  • Horizontal line tag (hr): This is used to draw an horizontal line that is used to divide the page into sections.
  • Strike tag (s): This is used to strike through text.
  • Height tag (height): This is used to determine the height of a table or image.
